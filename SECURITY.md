# Security Policy

MaintainerKit AI takes security and responsible open-source maintenance seriously.

---

## Supported Versions

The latest public release receives security updates and maintenance improvements.

| Version | Supported |
|---|---|
| v0.1.x | Yes |

---

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

Avoid publicly disclosing security issues before they are reviewed.

---

## Security Goals

MaintainerKit AI aims to:

- Maintain safe automation workflows
- Protect developer environments
- Encourage responsible AI usage
- Follow open-source security practices

---

## Maintainer Response

Security reports will be reviewed and prioritized based on severity and impact.
